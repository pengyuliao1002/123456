## 程式介紹

21點是一款紙牌遊戲，目標是在不超過21點的前提下，比對手的點數更高。

## 程式規則

遊戲規則為不超過21點的前提下，點數高的玩家獲勝。

## 程式玩法

遊戲以一個玩家和一個電腦進行，玩家先輸入名稱後開始遊戲。

## 程式碼安裝執行

程式使用Visual Studio開發，請按照以下步驟安裝和執行程式：

1. 下載並安裝Visual Studio。
2. 開啟Visual Studio並建立新的專案。
3. 複製程式碼至專案中的適當檔案。
4. 編譯並執行程式。
